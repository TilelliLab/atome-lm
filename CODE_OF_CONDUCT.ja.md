# 行動規範

本プロジェクトは [Contributor Covenant、バージョン 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) に従います。

要するに：敬意を持ち、善意を前提とし、議論を作業そのものに集中させてください。背景や属性にかかわらず、誰でも歓迎して貢献できます。容認できない行動の報告は **hello@atomelm.com** に非公開で送信でき、迅速かつ秘密裏に審査されます。

Contributor Covenant の全文——完全な施行ガイドラインを含む——は上記リンクにあり、本プロジェクトに全面的に適用されます。
